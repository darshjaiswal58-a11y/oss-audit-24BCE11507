## Project Description
This project focuses on analyzing open-source software and implementing Linux shell scripts to understand system operations and automation.

## Scripts Overview

### Script 1: System Identity Report
Displays system details such as OS, kernel version, user, uptime, and license.

### Script 2: FOSS Package Inspector
Checks whether Python is installed and shows its version and details.

### Script 3: Disk and Permission Auditor
Analyzes system directories for size, ownership, and permissions.

### Script 4: Log File Analyzer
Counts occurrences of keywords in system log files and displays results.

### Script 5: Open Source Manifesto Generator
Generates a personalized manifesto based on user input and saves it to a file.
